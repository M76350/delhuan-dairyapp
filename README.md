# 🥛 Delhuan Dairy - Milk Delivery App

**Delhuan Dairy** is a React Native application built with Expo, designed to provide seamless **milk delivery services** to customers. This app allows users to access fresh dairy products, schedule deliveries, and manage their milk subscriptions easily.

---

## 🚀 Features

- 🐄 Welcome & Home screens
- 📦 Get Started and View More navigation
- 📱 Mobile-first responsive design
- 🧭 React Navigation integration
- 🖼️ Image assets and screenshots included
- ☁️ Expo-based development and testing

## ▶️ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/delhuan-dairy.git
cd delhuan-dairy
````

### 2. Install dependencies

```bash
npm install
```

### 3. Start the project with Expo

```bash
npx expo start
```

Use `Expo Go` app on your phone to scan the QR code and run the app.

---

## 📦 APK / Deployment

* This project is built using **Expo managed workflow**
* You can generate a `.apk` file using:

```bash
npx eas build -p android --profile preview
```
